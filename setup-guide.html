‚úÖ Why Google Sheets?
Google Sheets provides a free, centralized database that works from any device. All player registrations will be stored in one place, accessible to admins from anywhere!

1Create Google Sheet

  1. Go to Google Sheets
  2. Click "+ Blank" to create a new spreadsheet
  3. Name it: "Cricket Registration Database"
  4. In the first row, add these column headers:
A1: ID | B1: Name | C1: Age | D1: Contact | E1: Email | F1: Place of Posting | G1: Address
H1: Role | I1: Batting Style | J1: Bowling Style | K1: Experience | L1: Jersey Size | M1: Status | N1: Registered At

2Create Apps Script

  1. In your Google Sheet, click Extensions ‚Üí Apps Script
  2. Delete any existing code in the editor
  3. Copy the code from google-apps-script.js file
  4. Paste it into the Apps Script editor
  5. Click File ‚Üí Save (or Ctrl+S)
  6. Name the project: "Cricket Registration API"
‚ö†Ô∏è Important: Make sure to copy the ENTIRE code from the google-apps-script.js file!

3Deploy as Web App

  1. Click Deploy ‚Üí New deployment
  2. Click the gear icon ‚öôÔ∏è next to "Select type"
  3. Choose "Web app"
  4. Configure settings:
    • Description: Cricket Registration API
    • Execute as: Me (your email)
    • Who has access: Anyone
  5. Click Deploy
  6. Click Authorize access
  7. Choose your Google account
  8. Click Advanced ‚Üí Go to Cricket Registration API (unsafe)
  9. Click Allow
  10. üìã COPY THE WEB APP URL - You'll need this!
‚úÖ Success! Your Web App URL should look like:
https://script.google.com/macros/s/AKfycby.../exec

4Update Your Website Code

  1. Open sheets-api.js in your project folder
  2. Find this line (near the top):
const GOOGLE_SHEETS_URL = 'YOUR_GOOGLE_SCRIPT_URL_HERE';
  1. Replace 'YOUR_GOOGLE_SCRIPT_URL_HERE' with your Web App URL
  2. Save the file
// Example:
const GOOGLE_SHEETS_URL = 'https://script.google.com/macros/s/AKfycby.../exec';

5Test the Integration

  1. Open register.html in your browser
  2. Fill out and submit a test registration
  3. Check your Google Sheet - the registration should appear!
  4. Open admin.html and login
  5. Click "Sync from Google Sheets"
  6. You should see the test registration in the admin dashboard
üéâ Congratulations! Your registration system is now connected to Google Sheets!

6Deploy Online

Now that everything is set up, you can deploy your website:

  1. Upload all files to your hosting service (GitHub Pages, Netlify, etc.)
  2. Test registration from different devices
  3. Verify admin can see all registrations
  4. Use "Download GitHub Backup" to save player data

üîí Security Notes

  • The Web App URL is public but only accepts data in the expected format
  • Regularly backup your Google Sheet data
  • Use the "Download GitHub Backup" feature to save player data
  • Keep your admin credentials secure
