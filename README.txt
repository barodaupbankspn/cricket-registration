DEPLOYMENT INSTRUCTIONS
=======================

1. Unzip this folder.
2. Upload ALL the files inside this folder to your hosting provider (e.g., Netlify, GitHub Pages).
   - IMPORTANT: Do not upload the folder itself, upload the *contents* of the folder.
   - Ensure 'index.html', 'dashboard.html', 'styles.css', etc., are in the root directory of your site.

TROUBLESHOOTING
===============
- If you see "Page Not Found" when clicking buttons:
  - Check if 'dashboard.html' and 'admin.html' were uploaded correctly.
  - Ensure file names are exactly as they appear (lowercase).

- If you see "Error: API helper not loaded":
  - Check if 'sheets-api.js' was uploaded.

- If you see "Error: Failed to fetch":
  - This might be a connection issue or the Google Sheet script is not accessible.
